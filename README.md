Chat Bidireccional con Sockets en Python

Este proyecto implementa un chat bidireccional usando sockets en Python. El objetivo es aplicar conceptos de Programación Orientada a Objetos (POO) y comunicación cliente-servidor de forma simple y funcional.

El chat permite que tanto el servidor como el cliente puedan enviar y recibir mensajes simultáneamente. Para lograrlo, se utilizan hilos (threading).

Características del proyecto

Comunicación cliente-servidor mediante sockets TCP.

Chat bidireccional en tiempo real.

Uso de Programación Orientada a Objetos:

Clase ServidorChat

Clase ClienteChat

Uso de hilos para evitar bloqueo en la recepción y envío de mensajes.
